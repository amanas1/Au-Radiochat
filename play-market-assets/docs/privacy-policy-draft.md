# Privacy Policy Draft for Au-Radiochat

Last updated: 2026-04-24

Au-Radiochat provides access to live radio streams and interactive listening features.

## Data We May Process
- Basic app usage data
- Radio preferences and favorites stored locally on device
- Optional profile data entered by the user
- Optional microphone or camera access only when a user actively uses features that require them

## Permissions
- Microphone: used only for user-triggered audio features
- Camera: used only for user-triggered visual or profile features
- Network access: required to stream radio stations and load app content

## Data Sharing
- Radio stream content may be delivered from third-party radio sources
- We do not claim ownership of third-party station content
- We do not sell personal data

## Local Storage
The app stores some settings locally on the device, such as favorites, profile preferences, and interface settings.

## Children
The app is not intended for children under the minimum age required by local law without parental supervision.
